# pluralkit-time
A library for getting fronting information from pluralkit, including who's fronting and how long it's been since someone fronted

### apikeys.json example
```{
    "pluralkit": 
    {   
        "token": "",
        "systemID": ""
    },
    "discord":
    {
        "token": "",
        "serverID": "",
        "userID": ""
    }
}```
